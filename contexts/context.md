# 项目上下文 - mypopku 宣传内容选题库

## 项目概述
本项目旨在为 `mypopku.com` 构建一个内部使用的宣传内容选题管理工具。该工具将帮助团队记录、分类和追踪宣传内容的创意选题。

## 技术栈
- **前端**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **持久化**: 浏览器 `localStorage` (初期方案)
- **设计风格**: 现代、高端、极简

## 核心场景
1. **爆款内容复刻**: 追踪小红书/即刻等平台的爆款（如 Jan 生成的互动动画），利用代码或工具进行技术复刻。
2. **节日专题策划**: 针对即将到来的节日（如元旦、新年）提前策划具交互性的贺卡或动画。
3. **团队协同跟进**: 多人共同维护选题库，标注负责人，追踪从“策划”到“发布”的全流程。

## 关键功能需求
- **置顶 (Pin)**: 快速定位核心选题。
- **状态追踪**: 精细化跟进完成情况。
- **筛选统计**: 按紧急程度或制作阶段筛选任务。
- **多人协作模拟**: 虽为单机演示，需预留团队协作的交互感知。

预置 5 个针对 `mypopku.com` 的运营选题。
